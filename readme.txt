== Description ==
 
Link Monitor is a FREE WordPress plugin that will help you to hide URLs from posts/pages and comments.

Example:
Original Link: AnOrdinaryWebsite/index.php?refId=234
Hidden Link: YourGreatWebsite/?goto=aHR0cDovL2Rkd2ViLmV1

Link Monitor works with custom posts, can be disabled for all or for choosen pages/posts.

== Installation ==

1. Upload to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all. Write posts with links
